Welcome to the Data Processing Dashboard

This project demonstrates automated data processing using Python and Pandas, integrated with GitHub Actions for continuous integration and deployment. The core logic processes an Excel file (`data.xlsx`), converts it to CSV (`data.csv`), and generates a structured JSON output (`result.json`) which is then published via GitHub Pages.

Explore the repository to understand the implementation details, including the Python script (`execute.py`), data handling, and the CI/CD pipeline. The latest `result.json` can be accessed directly through the navigation menu.

View on GitHub

How it Works

  • Data Source: An Excel file, data.xlsx, serves as the input.
  • Processing Script: execute.py reads data.xlsx, converts it to data.csv, and performs data manipulation.
  • Output: The script generates result.json containing processed data.
  • Automation: GitHub Actions automates linting (Ruff), execution of execute.py, and deployment of result.json to GitHub Pages.
